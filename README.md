## Main Features

- **Easy 1-click setup for Xenia** with automatic profile creation
- **Automatic updater for Xenia** (configurable via settings)
- **Easy Content Installer & Manager** – Install DLC/TUs without launching Xenia
- **Support for [Canary](https://github.com/xenia-canary/game-patches) & [Netplay](https://github.com/AdrianCassar/Xenia-WebServices/tree/main/patches) game patches**
- **Advanced Patch Management** – Add, edit, and remove patches in a unified window with duplicate patch support
- **Per-game configuration profiles** (with [community-optimized settings](https://github.com/xenia-manager/optimized-settings))
- **Xenia Profile Management** – Import, export, and edit profiles with automatic save backups
- **Dynamic Xenia Settings UI** – Automatically adapts to Xenia's config file structure
- **Smart Steam Shortcuts** – Creates shortcuts with full artwork support
- **Cross-platform Input Listener** – Reliable keyboard/mouse input detection using Avalonia
- **Import and export game saves** with XUID-based backup system
- **Low resource usage** with on-demand loading and caching
